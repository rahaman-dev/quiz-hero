1. What is different on map, filter, forEach, find

map, filter, find, and forEach are all array methods in JavaScript that are used to manipulate arrays. However, they each have different purposes and functions.

1. map method creates a new array with the results of calling a provided function on every element in the original array. It returns a new array of the same length as the original array.

2. filter method creates a new array with all elements that pass the test implemented by the provided function. It returns a new array with only the elements that satisfy the condition.

3. find method returns the value of the first element in the array that satisfies the provided testing function. It returns only the first element that satisfies the condition, and if none is found, it returns undefined.

4. forEach method executes a provided function once for each array element. It doesn't return anything and simply executes the provided function on each element in the array.

2. What is different on localStorage, sessionStorage, cookies

localStorage, sessionStorage, and cookies are all mechanisms in web browsers that are used to store data locally on the client-side. However, they differ in terms of their storage capacity, expiration, and accessibility.
1. localStorage: localStorage is a key-value storage mechanism that allows web developers to store data on the client-side that persists even after the user closes the browser. The data stored in localStorage can be accessed by any page from the same domain. In JavaScript, an event loop is a mechanism that enables asynchronous execution of code. The event loop constantly checks the queue for any new events, and if there are any, it executes them one by one in a loop.

To work with the event loop in JavaScript, you can use the setTimeout() and setInterval() functions. Here's a basic example of how you can use them:

// Define a function that will be executed after a certain time period function myFunction() { console.log("Hello world!"); } // Call the function after 2 seconds setTimeout(myFunction, 2000); // Define a function that will be executed repeatedly after a certain time period function myRepeatedFunction() { console.log("Hello again!"); } // Call the function every 1 second setInterval(myRepeatedFunction, 1000);

In this example, setTimeout() is used to execute the myFunction() function after a delay of 2 seconds. setInterval() is used to execute the myRepeatedFunction() function every 1 second.
